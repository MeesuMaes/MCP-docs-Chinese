# 贡献 MCP 文档

感谢您对贡献模型上下文协议 (MCP) 文档感兴趣！本指南将帮助您开始贡献。

## 开始之前

1. 安装所需的工具：
   - Node.js
   - Git
   - [Mintlify CLI](https://www.npmjs.com/package/mintlify): `npm i -g mintlify`
2. Fork 并克隆存储库
3. 为您的更改创建一个新分支
4. 运行 `mintlify dev` 在本地预览更改

## 文档指南

- 确保内容清晰、简洁且技术上准确
- 遵循现有的文件结构和命名约定
- 在适当的地方包含代码示例
- 使用正确的 MDX 格式和组件
- 测试所有链接和代码示例

## 内容类型

您可以贡献以下内容：

- 概念性指南
- API 参考文档
- 教程与快速入门
- 最佳实践与示例
- 故障排除指南

## Pull Request 流程

1. 确保您的更改符合我们的文档指南
2. 如有必要，更新相关目录
3. 使用 `mintlify dev` 在本地测试您的更改
4. 提交带有明确描述的 Pull Request
5. 等待审查并解决任何反馈
